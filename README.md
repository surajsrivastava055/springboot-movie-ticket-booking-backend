<h1>🎬 Movie Ticket Booking Application (Spring Boot)</h1>
<h4>A backend Movie Ticket Booking REST API built using Spring Boot, Spring Data JPA, and MySQL. 
This application allows users to manage movies, screens, shows, seats, and book movie tickets through RESTful APIs using a layered architecture.</h4>
<h2>🚀 Features</h2>
<b>
<or>
<li>Movie management (Add, update, delete, view movies)</li>
<li>Screen and show management</li>
<li>Seat management for different screens</li>
<li>Movie ticket booking system</li>
<li>Seat availability handling</li>
<li>Payment management for bookings</li>
<li>Global exception handling</li>
<li>RESTful API design with layered architecture</li>
<li>Database integration with MySQL</li>
</or>
</b>
 <h2> Tech Stack </h2>
<b>Java | Spring Boot | Spring Data JPA | REST API | Maven | MySQL </b></br>
